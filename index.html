🏗️ Defining Systems Before You Start

🏃
Player Movement
Handles movement only - not scoring
⭐
Collectibles
Manages pickup behavior only
🎮
Game Manager
Tracks orbs & win conditions
📱
UI System
Displays info - doesn't calculate
🔧 One System = One Responsibility
Separation of concerns makes each system easier to understand, test, and change.
